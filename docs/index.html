What it is

Blockwright lets an AI assistant operate Minecraft through controlled tools.

It is meant for local Minecraft Java Edition play first: ask inside the world, let the assistant inspect context and choose tools, then let the Fabric mod execute and verify changes through server-side APIs.

Talk from anywhere

Use /bw, Web, voice, Matrix/DingTalk, or local commands as different entrances to the same Minecraft assistant.

Fabric local worlds

The main path is a Java Edition/Fabric mod for single-player saves and LAN-opened worlds, so maps do not need to move to a Paper server.

Act through tools

The assistant can read player and world state, enqueue controlled actions, run supported commands, place blocks, and save blueprint records.

Edit safely

For existing builds, Blockwright scans nearby world blocks and matches saved build records before changing the structure.

Supported models

Choose the AI backend from the Web settings page.

Blockwright separates the Minecraft execution layer from the model provider. Configure the provider locally, then the same /bw, Web, voice, and chat-tool entrances use that assistant.

Codex CLI

Local-first assistant path for development and Minecraft tool workflows.

OpenAI

Use an OpenAI API key configured in the local Web settings.

DeepSeek

Alternative hosted model provider for chat and tool planning.

Doubao

Volcengine/Doubao model option for Chinese-friendly deployments.

Gemini

Google Gemini backend option for assistant requests.

Commands and entrances

Use /bw in game, or send the same request from Web and chat tools.

The command surface stays small. Natural language carries the task, while Blockwright decides which controlled tool or action should run.

/bw <request>

Main Minecraft command

Send a natural-language request such as scan what I am looking at, set the time to day, give me torches, or replace this wall with stone bricks.

/bw web

Open Web UI

Print the local Web address in Minecraft chat so you can configure the player name, model provider, text chat, and voice input.

Web text and voice

Browser entrance

Send text from a browser, or hold the microphone button to speak. Requests are sent to the Minecraft player configured on the page.

Matrix, DingTalk, local scripts

Chat-tool entrances

Connect external chat tools or local command scripts when you want Minecraft requests to come from outside the game window.

How to use

Install the Fabric mod, choose a model, then send a request.

The player flow stays simple: prepare the Fabric profile, put the required mods in the right folder, and use /bw, Web, voice, or a connected chat tool.

Prepare Minecraft

Use Minecraft Java Edition 1.21.x with Fabric Loader 0.16.14 or newer.

Put mods in mods/

Install Fabric API for your 1.21.x version, then put blockwright-fabric-*.jar in the same mods/ folder.

Choose an AI model

Run /bw web, open the Web page, set your Minecraft username, and choose Codex CLI, OpenAI, DeepSeek, Doubao, or Gemini.

Send a request

Use /bw, Web text, voice, or a connected chat tool. Try /bw scan what I am looking at and make this wall stone brick.
