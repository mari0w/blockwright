What it does

Blockwright turns Minecraft requests into controlled in-game actions.

It is meant for local Minecraft Java Edition play first: ask inside the world, let the assistant plan, and let the Fabric mod place or verify the result through server-side APIs.

Talk in game

Use /bw to ask for items, time changes, builds, or edits without leaving the Minecraft world.

Fabric local worlds

The main path is a Java Edition/Fabric mod for single-player saves and LAN-opened worlds, so maps do not need to move to a Paper server.

Build with records

Build plans are saved as blueprints and records before the same block list is sent to Minecraft for placement.

Edit safely

For existing builds, Blockwright scans nearby world blocks and matches saved build records before changing the structure.

Try it locally

Install the mod, open Minecraft, then talk to Blockwright.

The player flow is intentionally simple: prepare the Fabric profile, put the required mods in the right folder, and use /bw or the Web page.

Prepare Minecraft 1.21.x

Use Minecraft Java Edition 1.21.x with Fabric Loader 0.16.14 or newer.

Put mods in mods/

Install Fabric API for your 1.21.x version, then put blockwright-fabric-*.jar in the same mods/ folder.

Open Minecraft and use /bw

Run /bw web to open the Web page, then try /bw give me a diamond sword.
