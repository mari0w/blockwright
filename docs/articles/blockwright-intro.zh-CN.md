# Blockwright：让 Minecraft 玩家用一句话完成游戏操作

如果你经常玩 Minecraft Java 版，大概会遇到这样的时刻：

- 想快速拿一件物品，但不想翻命令表；
- 想盖一个小木屋、房间或树屋，但不想从第一块木板慢慢摆；
- 想把眼前这面墙换成玻璃，或者把房子的窗户换成蓝色玻璃；
- 想切到白天、停止下雨、换个游戏模式；
- 想让朋友在局域网世界里也能直接通过聊天发需求。

Blockwright 做的事情很简单：把这些 Minecraft 需求变成一句自然语言指令。

你可以在游戏里输入：

```text
/bw 给我一把钻石剑
/bw 帮我盖一个带窗户和床的小木屋
/bw 把时间调到白天
/bw 把我面前这面墙换成玻璃
```

然后 Blockwright 会把玩家的意图交给 AI 助手理解，再通过 Fabric 模组在 Minecraft 世界里执行。

## Blockwright 是什么？

Blockwright 是一个面向 Minecraft Java 版玩家的 AI 助手。

它的主安装方式是 Fabric 模组，适合 Java 版 1.21.x、单人存档和局域网开放世界。玩家不需要把原来的地图迁移到 Paper 服务端，也不需要为了使用 AI 助手改变自己的游玩方式。

装好后，你可以通过几种方式和它说话：

- 在 Minecraft 聊天栏里输入 `/bw ...`
- 打开 Web 页面直接打字
- 在 Web 页面按住麦克风说话
- 通过 Element/Matrix 房间发消息
- 通过钉钉机器人发消息
- 通过本地命令或脚本接入其他工具

它不是一个只会聊天的机器人。它的目标是把聊天变成 Minecraft 世界里的实际动作。

## 它能做什么？

### 1. 发物品

你可以直接说：

```text
/bw 给我一组火把
/bw 给我一套钻石装备
/bw 给我一把带击退的剑
```

这类需求适合替代常见的 `/give` 命令。玩家不用记物品 ID，也不用手写完整命令。

### 2. 调整游戏状态

你可以说：

```text
/bw 把时间调到白天
/bw 别下雨了
/bw 给我夜视
/bw 切到创造模式
```

这些都是玩家在建造、测试或局域网游玩时经常会用到的操作。

### 3. 建造小型建筑

Blockwright 可以根据自然语言生成建筑需求，例如：

```text
/bw 帮我盖一个小木屋
/bw 在我面前建一个带窗户和床的房间
/bw 建一个简单的树屋，入口要能走上去
```

它适合处理小屋、房间、树屋、平台、简单装饰和一些局部结构。你仍然可以自己继续修改，但第一版不必从空地开始。

### 4. 改造眼前的东西

除了新建，它也可以处理一些“改造”类需求：

```text
/bw 把这个房子的窗户换成蓝色玻璃
/bw 把我面前这面墙换成石砖
/bw 把屋顶加高一点
```

这种交互更接近“你在游戏里指着一个地方，然后告诉助手要怎么改”。

### 5. 继续对话

Blockwright 会围绕当前玩家和上下文继续处理需求。你可以连续说：

```text
/bw 帮我盖一个小木屋
/bw 换大一点
/bw 加几盏灯
/bw 把屋顶换成深色木头
```

这种方式比一次性写完整需求更自然，也更接近真实玩家的建造流程。

## 怎么开始使用？

玩家侧的流程尽量保持简单：

1. 准备 Minecraft Java Edition 1.21.x。
2. 安装 Fabric Loader 0.16.14 或更新版本。
3. 在当前游戏目录的 `mods/` 文件夹里放入：
   - 对应 1.21.x 游戏版本的 Fabric API
   - `blockwright-fabric-*.jar`
4. 启动 Minecraft，进入世界。
5. 在游戏聊天栏输入：

```text
/bw web
```

打开显示出来的 Web 地址后，填写自己的 Minecraft 用户名，再选择一个 AI 模型。

目前支持：

- Codex CLI
- OpenAI
- DeepSeek
- 豆包 Doubao
- Gemini

配置完成后，就可以在 Web 页面、语音、游戏内 `/bw` 或聊天工具里发送需求。

## 为什么这件事有意思？

Minecraft 的魅力在于自由创造，但很多重复操作并不一定有趣。

拿物品、调时间、清雨、搭一个基础小屋、替换一片墙、给房间补灯，这些事情都很常见。Blockwright 想做的是把这些重复操作变成一句话，让玩家把注意力放回想象和设计上。

它适合：

- 单人存档里快速试想法；
- 局域网世界里让朋友通过聊天发需求；
- 创造模式里快速搭出第一版建筑；
- 服主或运营把 Minecraft 接到 Web、语音或聊天工具里；
- 想体验“AI 参与 Minecraft 游玩”的玩家。

## 一句话总结

Blockwright 不是要替你玩 Minecraft。

它更像一个能听懂自然语言的 Minecraft 助手：你负责提出想法，它负责把一部分重复、繁琐、命令化的操作变成游戏里的实际动作。

如果你想让 Minecraft 里的建造、发物品、改造和调试更像“说一句话就能开始”，Blockwright 就是为这个方向做的尝试。
