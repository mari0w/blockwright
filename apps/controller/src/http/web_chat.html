Mobile HTTPS

The controller generates a local root certificate and LAN HTTPS address. Install and trust the certificate before using voice on a phone.

HTTPS address:

Download Blockwright local root certificate

On iPhone/iPad, open the certificate link in Safari. Install the downloaded profile from Settings, then enable full trust in Certificate Trust Settings.

On Android, saving through Files by Google or the file manager only downloads the certificate. Install it manually from Security or Privacy > Encryption and credentials > Install certificate > CA certificate.

Android usually will not prompt again after the file is saved. Manually install the CA certificate, then reopen the HTTPS address and request microphone permission.
